## Build

### dependency

- CMake
- Vulkan SDK >= 1.3.283
- X11 development package(only on Linux)

### Command

```
git clone https://github.com/Kiterai/BrightCpp.git
cd BrightCpp
cmake . -B build
cmake --build ./build
```
